# microriv-soc
A simulation-first RISC-V mini SoC built from scratch — integrating an open-source RISC-V core with custom UART, GPIO, and Timer IPs over an APB bus. Features multi-clock domain architecture with verified CDC/RDC synchronization, async FIFOs, clock/power gating via UPF, and SVA-based CDC assertions. Verified using Verilator.
